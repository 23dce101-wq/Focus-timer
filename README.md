## About

Focus Timer is a habit tracking and productivity application built using modern web technologies.  
It allows users to create goals, visualize analytics, and manage daily routines efficiently.

---

## Features

- Habit Tracker  
- Productivity Timer  
- Visual Analytics and Dashboards (charts, statistics)  
- Backend API for habit and metric storage  
- Live deployment via Vercel

## Installation

You need **Node.js and npm** installed:

1. Clone the repository:
```bash
git clone https://github.com/23dce101-wq/Focus-timer.git

Navigate to Frontend:

cd Frontend
npm install


Navigate to Backend:

cd ../backend
npm install

Run Frontend
cd Frontend
npm run dev

Run Backend
cd backend
npm start

Deployment

This project is deployed on Vercel. It automatically builds from GitHub pushes.

You can check the live deployment here:
https://focus-timer-flame.vercel.app/
